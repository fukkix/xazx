## 待开发功能 📋

### P1: Excel 复制粘贴直接生成表格
**价值**：运营人员最常用的 workflow，无需先保存文件再导入。  
**预期行为**：
1. 在 Excel 中复制选区
2. 回到编辑器画布 `Ctrl + V` 粘贴
3. 自动识别剪贴板 `text/html` 或 `text/plain`（制表符分隔）
4. 在光标位置生成一个 `TableBlock`

**状态**：✅ 已完成（支持 HTML 表格和制表符分隔文本自动识别合并单元格）

### P1: 新增 FeatureMatrixBlock（功能矩阵专用块）
**价值**：针对功能清单表格的最优长期方案，比通用 TableBlock 效率高一个数量级。  
**预期行为**：
- 数据模型：`categories[] → features[] → supports[version]`
- 类别列自动跨行合并
- 支持一键添加/删除版本列
- 支持批量勾选 √ / ×
- 类别可折叠

**状态**：✅ 已完成（支持类别折叠、版本列增删/重命名、单元格双击循环切换标记、列批量填充）

### P2: 表格行分组折叠 + 列冻结
**价值**：百页长表格的浏览体验质变。  
**预期行为**：
- 行分组：选中多行 → 设为分组 → 自动合并首列 rowspan
- 分组左侧出现 ▶/▼ 折叠箭头
- 列冻结：首列（功能类别）和第二列（功能项）横向滚动时固定

**状态**：✅ 已完成（TableBlock 展示态支持自动识别 rowspan 分组并折叠；支持冻结前 1/2 列）

### P2: 图片块草稿持久化增强
**价值**：解决大文档图片在草稿恢复后丢失的问题。  
**预期行为**：
- 将图片 File 转为 Base64 存入 localStorage（限制单张 < 500KB 或总大小 < 4MB）
- 恢复草稿时自动还原为 File 对象和 Blob URL

**状态**：✅ 已完成（saveDraft 时自动将图片 File 编码为 base64 存入 attrs；loadDraft 时自动还原为 File + Blob URL）

### P3: 段落级评论批注
**价值**：协作审阅。  
**预期行为**：
- 选中段落文字 → 添加批注
- 批注气泡显示在块右侧
- 数据模型已预留，需 UI 实现

**状态**：⏳ 待开发
